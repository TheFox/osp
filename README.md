# One Shall Pass for Command-line

## Install

The preferred method of installation is via RubyGems.org:  
<https://rubygems.org/gems/osp>

```bash
gem install osp
```

or via `Gemfile`:

```ruby
gem 'osp', '~>0.6'
```
